## ✨ Features

- **Modern UI/UX**: Clean, professional design using a soft purple and white color palette with glassmorphism elements.
- **Interactive Dashboard**:
  - Real-time dynamic statistics cards (Total Students, Active Students, Departments, Courses).
  - Modern student records table with sleek active badges and action buttons.
- **Add Student Functionality**: Click the **"+ Add New Student"** button to open a smooth modal form, append new students to the table, and increment statistics automatically.
- **Live Search**: Instantly filter the student records table by typing in a Student Name, Roll Number, or Email Address.
- **Delete Functionality**: Remove student records dynamically with the trash-can icon, which automatically decrements the live statistics.
- **Responsive Layout**: Designed to adapt perfectly to both desktop and mobile devices.

## 🚀 Built With

- **Laravel** (Backend Structure)
- **HTML5 & CSS3** (Vanilla CSS, custom CSS Variables, Flexbox/Grid layout)

## 🛠️ Setup & Run

To preview the project locally:

1. Clone the repository.
2. Ensure you have PHP and Composer installed, then run:
   ```bash
   composer install
   ```
3. Start the Laravel development server (using Laravel Herd or Artisan):
   ```bash
   php artisan serve
   ```
4. Visit the local URL provided (usually `http://127.0.0.1:8000`) in your browser to view the Student Record Management System dashboard.
